# Discord YenaBot Source 

## 개요
- 기존에 서비스하던 디스코드의 예나봇을 오픈소스로 공개합니다.

- 아래 라이선스에 따라 사용한다면, 자유롭게 사용이 가능합니다.

## 사용 언어
- 이 예나봇에는 NodeJS와 DiscordJS가 사용되었습니다. 

- 자바스크립트로 작동되는 봇입니다.

## 운영 서버와의 호환성
- 우분투 18.04 버전에서 구동하였지만, 윈도우에서도 구동이 가능합니다.

- 하지만 우분투에서 구동하는 것을 여러므로 추천드립니다.
